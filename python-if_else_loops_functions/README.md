# Python - if, else, loops, functions

## Description
This project is part of the Holberton School curriculum and focuses on the basics of Python programming. It covers fundamental concepts such as conditional statements (`if`, `else`, `elif`), loops (`while`, `for`), and the definition and use of functions. Additionally, it explores character manipulation using ASCII values.

## Requirements
* **Allowed editors:** `vi`, `vim`, `emacs`.
* **Environment:** All files are interpreted/compiled on Ubuntu 20.04 LTS using `python3` (version 3.8.5).
* **Style:** All code must follow the `pycodestyle` (version 2.8.*) guidelines.
* **Execution:** All files must be executable.
* **Format:** All files must end with a new line and start with `#!/usr/bin/python3`.
