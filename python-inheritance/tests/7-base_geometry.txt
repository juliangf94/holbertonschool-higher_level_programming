====================================
How to use the BaseGeometry class
====================================

First, we import the class from the module:
>>> BaseGeometry = __import__('7-base_geometry').BaseGeometry
>>> bg = BaseGeometry()

-------------------------
Testing integer_validator
-------------------------

The method integer_validator(name, value) validates that 'value' is a positive integer.
'name' is assumed to be a string.

Valid cases (should not raise any exception):
>>> bg.integer_validator("my_int", 12)
>>> bg.integer_validator("width", 89)
>>> bg.integer_validator("height", 1)

Invalid type - Not an integer (raises TypeError):
>>> bg.integer_validator("name", "John")
